# vue-project

Vue CLI 3 / Vue 2 프로젝트. 최신 Node(예: `nvm use stable`)에서 돌리려면 저장소에 맞춰 둔 `serve`/`build` 스크립트(OpenSSL legacy + dart-sass)를 사용합니다.

## Project setup

```
nvm use   # vue-project/.nvmrc → stable
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
